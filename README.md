# 和小波聊聊

一个让用户与“受王小波作品中的理性、幽默、自由与反讽精神启发的虚拟对话者”交流的非官方文学互动网站。

## 项目边界

- 不是王小波官方项目，不代表王小波本人。
- 不制造或传播虚假语录。
- 站内对话与收藏句子统一标注为 AI 生成内容。
- 不复制受版权保护的作品长段原文。

## 当前功能

- 沉浸式首页与响应式聊天界面
- 清醒、荒诞、温和、辩论四种模式
- 本地保存对话、收藏句子、删除与纯文本导出
- 随机问题、今日一问、演示对话
- 分享图片卡片、写给未来的信、荒诞诊断书
- 阅读入口、句子档案、关于本站
- 浅色与深色模式
- 可替换的服务端 AI 接口适配层

当前公开演示使用本地模拟回复；接入真实模型时请将密钥保存在服务端环境变量中，切勿写入浏览器代码。

## 本地运行

```bash
pnpm install
pnpm run dev
```
